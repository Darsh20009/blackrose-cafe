🚫 أول حاجة: إلغاء فكرة الـ PDF تمامًا

اللي عندك دلوقتي:

النظام بيعمل PDF → بعدين يبعته للطباعة

❌ ده أكبر سبب للمشكلة

طابعات الكاشير (ESC/POS) ما بتفهمش PDF
النتيجة = ورق أبيض أو طباعة بايظة

✅ الحل:

اطبع Direct Raw Data (ESC/POS) مش PDF

✅ النظام الصح (اللي الكاشير الحقيقي بيستخدمه)

بدل:

Generate PDF → Print

خليها:

Generate ESC/POS commands → Send مباشرة على IP:9100
🧠 يعني إيه ESC/POS؟

ده بروتوكول أوامر للطابعات الحرارية (زي Epson)

مثال:

تكبير الخط
فتح الدرج
قص الورق
طباعة باركود
🔧 التنفيذ الفعلي (أهم جزء)
1️⃣ الاتصال بالطابعة (Socket TCP)

لازم تبعت Data مباشرة على:

IP: 192.168.8.77
Port: 9100
مثال (Java / Android):
Socket socket = new Socket("192.168.8.77", 9100);
OutputStream outputStream = socket.getOutputStream();
2️⃣ ابعت أوامر ESC/POS بدل PDF
مثال طباعة فاتورة:
outputStream.write(new byte[]{0x1B, 0x40}); // Initialize

outputStream.write("My Store\n".getBytes());
outputStream.write("--------------------------\n".getBytes());

outputStream.write("Item 1    10 SAR\n".getBytes());
outputStream.write("Item 2    20 SAR\n".getBytes());

outputStream.write("--------------------------\n".getBytes());
outputStream.write("Total:    30 SAR\n".getBytes());

outputStream.write(new byte[]{0x1D, 0x56, 0x41}); // Cut paper
3️⃣ اقفل الاتصال
outputStream.close();
socket.close();
⚙️ 3 حلول جاهزة (اختار واحد)
